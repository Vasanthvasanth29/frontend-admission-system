# Engineering Admission System (Frontend)

This is the frontend of the Engineering Admission System built using **React + Vite**.

## Features
- Student Registration
- Student Login
- View Available Engineering Courses
- Apply for Courses
- Check Admission Status
- Admin Panel for Managing Courses and Applications

## Technologies Used
- React
- Vite
- Axios
- CSS

## How to Run

1. Install dependencies

npm install

2. Start the development server

npm run dev

Frontend will run on:

http://localhost:5173

## Backend

Backend is developed using **Spring Boot + MySQL**.
